# 🏎️ F1 Pit Wall Dashboard

🔴 **Live Website:** https://f1-pitwall-bhavit.netlify.app/

A personalized **Formula 1 dashboard** built for **Bhavit**, featuring a Ferrari-inspired theme and a focus on **Charles Leclerc**.
This project showcases a modern UI with racing-style design, animations, and structured data display.

-## 🚀 Features

* 🔥 Ferrari-themed UI (red racing accents)
* 🧑 Personalized dashboard (Bhavit)
* 🏁 Driver & Constructor standings
* ⏱️ Race countdown timer
* 📅 F1 season calendar
* 📰 Paddock news section
* ⚡ Smooth animations & transitions
* 💻 Fully responsive design

## ⚙️ Setup & Usage

1. Clone the repository:

```bash
git clone https://github.com/nagabhavit/f1-pitwall-bhavit.git
```
2. Open the project folder:
```bash
cd f1-pitwall-bhavit
```
3. Run the project:
* Simply open `index.html` in your browser
